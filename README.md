# Akhil Bhalodiya — Portfolio

Terminal-style developer portfolio inspired by developer-portfolio by 1hanzla100.
Built with Next.js 15 + TypeScript.

## Deploy to Vercel (2 min)
1. Push this folder to a GitHub repo
2. Go to vercel.com/new → Import the repo
3. Click **Deploy** — zero config needed ✅

## Run locally
```bash
npm install
npm run dev
# http://localhost:3000
```

## Customise
- **LinkedIn URL**: search `akhil-bhalodiya` in all files and replace with your actual slug
- **GitHub URL**: replace `akhil-bhalodiya` with your GitHub username
- **Resume**: replace `public/resume.pdf` with an updated version anytime
- **Content**: all data is inline in each component under `/components/`
